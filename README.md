# Playwright CRUD Automation Framework

## Overview
Automated testing framework built using Playwright and TypeScript for OrangeHRM.

## Features
- Login Automation
- Add Employee
- Search Employee
- Update Employee
- Delete Employee
- HTML Reports

## Tech Stack
- Playwright
- TypeScript
- Node.js
- GitHub

## Run Tests

```bash
npx playwright test
```

## Generate Report

```bash
npx playwright show-report
```
